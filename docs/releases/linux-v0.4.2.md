# BioCSSwitch Linux linux-v0.4.2

BioCSSwitch Linux 0.4.2 brings the blue-and-white scientific workspace and a more efficient control layout to the Linux x64 edition.

## Highlights

- Blue, white, and light-blue-gray surfaces provide clearer research, configuration, and runtime hierarchy.
- Connection profiles are grouped on the left; runtime and advanced controls remain visible on the right.
- Research tools, privacy controls, MCP/Skill verification, and task routing use a responsive two-column grid.
- The Chinese and English README files now document Linux-native `.deb` and AppImage installation.

## Packages

- `BioCSSwitch_0.4.2_amd64.deb` for Ubuntu 22.04+ and Debian 12+.
- `BioCSSwitch_0.4.2_amd64.AppImage` for compatible x64 Linux distributions.

Install Claude Science for Linux first and ensure `claude-science` is available on `PATH`. See `docs/LINUX.md` for installation and source-build details.

BioCSSwitch is a research-assistance tool and does not provide medical advice. Researchers must review and verify all AI-assisted outputs.
