# BioCSSwitch bio-v0.4.1 — AI 生物医学研究平台

BioCSSwitch 0.4.1 将产品入口从“模型切换”重新定义为“研究工作流”。模型连接仍然可用，但已下沉到设置页；研究者打开应用后先选择要完成的生物医学任务。

## 研究工作台

- 新增任务优先首页：文献综述、单细胞分析、实验方案设计和跨模态靶点排序均可一键准备研究工作区。
- 每个入口会保存任务路由、按需启用相应本地工具包，并启动 Science 研究环境。
- 新增 `experimental-design` 任务路由，支持在任务级别设置模型连接。

## 研究者可见的输出

- 文献综述工作流突出声明级证据审计、GRADE Summary of Findings 和五段不确定性面板。
- 跨模态工作流突出来源、冲突、缺失模态和靶点排序，而不是单一模型生成的结论。
- 新增独立研究平台介绍页，展示证据审计链、GRADE/SoF、不确定性面板与跨模态排序的示例字段。

## 发布与安装

- 应用版本统一为 `0.4.1`。
- 推送 `bio-v0.4.1` 标签会在 macOS runner 构建 Apple Silicon DMG，并自动上传到同名 GitHub Release。
- 当前 DMG 为未公证构建；首次运行可能需要在 macOS 中右键选择“打开”。

## 重要说明

BioCSSwitch 是研究辅助工具，不提供医疗建议。AI 生成的证据综合、靶点排序和实验建议必须由合格研究者审阅、核验并在适当的研究治理下使用。
