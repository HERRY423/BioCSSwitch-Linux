# BioCSSwitch v0.3.3 Source-Release Waiver

Date: 2026-07-06

Scope: source release for `HERRY423/BioCSSwitch` v0.3.3. This waiver does not approve a macOS `.dmg` binary release.

## Passed Gates

- Gate 1, pack manifest and dependency graph: passed through `python test/test_bio_offline.py`.
- Gate 2, version consistency: passed through `python test/test_bio_offline.py`.
- Gate 6 coverage check: `python test/bio_eval/gold_calibration.py --check` passed.
- Additional benchmark self-test: `python test/bio_eval/run.py --selftest` passed with 70 cases.
- Frontend syntax: `node --check desktop/src/main.js` passed.

## Waived For This Source Release

- Gate 3, local Rust/Tauri build evidence: waived because this Windows environment has no `cargo`, and WSL/bash is unavailable. Binary packaging must not be attached until `bash scripts/release-verify.sh --build` passes on a macOS build host.
- Gate 4, real Claude Science canary: waived because no dated manual canary evidence JSON was available in this workspace. Treat pack path verification as experimental until a canary artifact passes `scripts/check-canary-evidence.py`.
- Gate 5, provider matrix: waived because no provider result files were present. `python test/bio_eval/provider_matrix_gate.py --min-providers 2 --min-repeat 3` reported 0 providers. Provider quality claims must not be made for this release.
- Gate 6 strict expert approval: waived for source release. `python test/bio_eval/gold_calibration.py --check --strict` reported `needs_expert_review` for `dr_metformin_mech` and `ea_batch_verify`. Expert-approved clinical/biomedical benchmark claims must wait for strict approval.

## Required Before Binary Release

1. Run `bash scripts/release-verify.sh --build` on macOS with Rust, Node, npm, and Tauri available.
2. Capture real Claude Science canary evidence and validate it with `python scripts/check-canary-evidence.py <evidence.json>`.
3. Run at least two providers with repeat >= 3 and pass `python test/bio_eval/provider_matrix_gate.py --min-providers 2 --min-repeat 3`.
4. Obtain expert approval for gold calibration entries, then pass `python test/bio_eval/gold_calibration.py --check --strict`.
